⌘ + P => 검색 파일 열기 

⌘ + X => 라인 삭제

⌘ + L => 커서 라인 전체 선택

⌘ + D =>  같은 단어 선택

⌘ + ] => 들여쓰기

⌘ + [ => 내어쓰기

⌘ + F = > 찾기

⌘ + ⌥ + F => 바꾸기

⌘ + ⇧ + F => 파일에서 찾기

⌘ + Up = > 문서 처음으로 커서 이동

⌘ + Down => 문서 마지막으로 커서 이동

⌘ + [NUM] = > 입력 탭으로 이동 1-9

⌘ + ⌥ + 1~5 => 화면 1~5단으로 보기

⌘ + / => 선택 주석
